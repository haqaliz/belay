## Phase 1 — 🔴 BTH-1 tree hash (RED first)

**Goal:** the measuring instrument. **Everything downstream is judged by it, so it comes first.**

- **RED:** `tests/test_bth1.py` — hash a fixture tree; assert deterministic across two runs.
- **The fixture is the deliverable** (`tests/fixtures/torture_tree.py`): regular files · dirs ·
  symlinks (rel + abs) · **hardlink pair** · **setuid file (`0o4711`)** · xattr · `st_flags`
  (`UF_HIDDEN`) · empty dir · **unicode NFD filename** · sparse file · nested depth.
  **NO FIFO/socket/device here** — those belong to Phase 5's refusal tests.
- **GREEN:** `src/belay/snapshot/bth1.py`.
  - **IN:** `BTH-1` tag · **raw readdir PATH BYTES** · kind · `S_IMODE` · `mtime_ns` · `st_flags` ·
    uid/gid · size + sha256 content (regular only) · symlink target **raw bytes** · `st_rdev` ·
    sorted `(xattr_name, sha256(value))` minus ignore-list · **hardlink GROUP ID = first path in
    group** (*not* the inode).
  - **OUT, documented:** inode/`st_dev` · **atime (self-invalidating)** · ctime · **birthtime** ·
    `st_blocks` · raw `nlink`.
  - **Sort by RAW PATH BYTES** (readdir order is not stable; `str` sort reintroduces the
    normalization trap).
  - `sha256("BTH-1\n" + concat(sha256(rec)))`; records `\x00`-joined `key=value` **so a failure names
    the field**.
  - **xattrs need `ctypes`** — `os.listxattr` **does not exist on macOS**.
  - **Ignore-list MUST include `com.apple.provenance`** — macOS injects it on every file, so a naive
    "xattrs present?" check passes on a tree that lost all real xattrs.
  - **Gate on `S_ISREG` before opening anything** — `open(fifo).read()` **blocks forever**.
- **A7 NEGATIVE CONTROLS, same phase:** no-op re-hash · **atime churn (read every file)** · inode
  churn → BTH-1 **MUST NOT move**. *Without these, a hash that always differs also "passes".*

**Validation:** `uv run pytest tests/test_bth1.py`
**Commit:** `Add the BTH-1 tree hash and the torture-tree fixture`

## Phase 2 — 🔴 Snapshot/restore: clonefile + the 3 repairs

**Goal:** A1 + A5. **The capability's core, and it is already measured working** (10.1ms/3.5ms).

- **RED:** `tests/test_snapshot.py::test_pre_state_restores_byte_identically` —
  snapshot → mutate → restore → `BTH-1(restored) == BTH-1(original)`. **Fails: no snapshot module.**
- **GREEN:** `src/belay/snapshot/clone.py`
  - `ctypes.CDLL(ctypes.util.find_library("c"), use_errno=True)`; `clonefile(src, dst, CLONE_ACL)`.
  - **`CLONE_ACL = 0x0004` is MANDATORY** — `flags=0` **silently drops ACLs**.
  - **Clones whole trees recursively in ONE call.**
  - **Do NOT shell out to `cp -Rc`** — it additionally recreates non-regular files fresh, losing
    symlink/FIFO mtimes.
  - **The 3 repairs, deepest-first:** relink hardlinks · re-chmod setuid (clonefile strips it:
    `0o4711 → 0o0711`) · restore dir mtimes.
- **A5 REPAIR-ABLATION, same phase, permanent:** disabling `hardlinks` / `suid` / `dirmtimes`
  (each, and all three) **MUST fail A1**; all on → PASS.
  > **This is the guard that stops someone "simplifying" a repair away and leaving a green suite.**

**Validation:** `uv run pytest tests/test_snapshot.py`
**Commit:** `Add clonefile snapshot/restore with the three sidecar repairs`

## Phase 3 — Mutation suite (A6) — field-attributed

**Goal:** prove A1 can fail, **for the right reason**.

`tests/test_snapshot_mutations.py` — 12 corruptions applied to a **perfect** copy, each asserted to
be caught **by the field under test**:
lost mode bit · **lost setuid** · symlink→copy-of-target · **symlink retargeted** · **NFD→NFC
filename** · dropped xattr · **hardlink broken into 2 inodes** · dropped `st_flags` · mtime ±1ns ·
dropped empty dir · fifo→regular file · 1-byte content change.

> 🔴 **The researcher's suite caught ITSELF passing for the wrong reason.** Several mutations were
> detected by the **parent dir's mtime**, not the property under test; *"symlink retargeted"* was
> caught by `d`'s mtime. After neutralizing dir mtimes it was **still** confounded by the symlink's
> own fresh mtime — only `os.utime(..., follow_symlinks=False)` isolated it.
> **Neutralize dir mtimes AND symlink mtimes in the harness, then assert the field.**
> **Two mutations are content-identical by construction** (symlink-deref, hardlink-break) and exist
> **purely to kill a content-only hash** — they must never be "simplified" away.

**Validation:** `uv run pytest tests/test_snapshot_mutations.py`
**Commit:** `Prove the restore test fails for the right reason: field-attributed mutations`

## Phase 4 — 🔴 Seatbelt: containment + denials (A2, A3, A8, A9)

**Goal:** the sandbox. **A2's traps are why this is a whole phase.**

- **RED:** `tests/test_containment.py` — escape vectors: direct · `../` · symlink-out · `mv`-out ·
  **grandchild**.
  **Each asserts THREE things**: no file · **exact rc (1, NOT 134, NOT 137)** · **exact stderr
  (`Operation not permitted`)**.
  > 🔴 **An over-strict profile SIGABRTs (rc=134) before the process acts** — a test asserting only
  > `not exists()` **PASSES against a sandbox that murders everything**.
- **A3 POSITIVE CONTROL, same phase, non-negotiable:** the same write **succeeds** when permitted;
  egress **succeeds** under allow-all. *Without it, an unplugged network passes the egress test
  forever.*
- **GREEN:** `src/belay/sandbox/seatbelt.py`
  - Profile: `(version 1) (deny default) (allow process*) (allow sysctl-read) (allow mach-lookup)
    (allow file-read*) (allow file-write* (subpath <scope>)) (deny network*)`.
    **`mach-lookup` is required or `/bin/sh` dies at launch.**
  - **M3: `os.path.realpath()` EVERY path before it reaches a profile.** `/tmp` → `private/tmp`;
    without this the profile **silently grants nothing** and denies the user's own work (A9).
  - **M2 network vocabulary is a CLOSED ENUM**: `deny-all | allow-all | allow-ports`.
    **A8: a hostname allowlist MUST be REJECTED at the API** — `(remote ip "1.1.1.1:*")` is a
    compile-time error, so accepting one would be claiming a boundary we don't enforce.
  - **M2: denials recorded** → append `kind: "denial"` via C1's `TraceWriter` (extensible kinds, no
    schema break). Record **the exact path** — M13: this makes widening a **diagnosis**, not a guess.
  - **M9: record `kind: "network_policy"` per turn as a FACT.** **C2 does not judge replayability;
    C4 does.**

**Validation:** `uv run pytest tests/test_containment.py`
**Commit:** `Add the Seatbelt sandbox: contained escapes, recorded denials`

## Phase 5 — Refuse out-of-substrate + the `unrestorable` signal (A4)

**Goal:** M7 + M8. **The honesty contract as code.**

- **RED:** a FIFO in the tree → **detected and REFUSED with a named cause**; `state_handle` =
  `{"status":"unrestorable","cause":...}`. **Never a silent skip, never a silent success.**
  > *"Narrowing the substrate is only honest if you detect out-of-substrate entries and refuse
  > loudly. Silently ignoring a FIFO is exactly the UNVERIFIED-rendered-as-PASS `CLAUDE.md`
  > forbids."*
- **GREEN:** `src/belay/snapshot/substrate.py` + fill C1's `state_handle` slot.
  Supported: **regular files, dirs, symlinks**. Everything else → named cause.
  Taxonomy: `understanding.md` §7.
- **S1:** `SnapshotBackend.capabilities()`; the manifest records **which backend produced it**;
  **restore REFUSES across differing capability sets.**
  > Honesty, not portability: `os.listxattr` doesn't exist on macOS but does on Linux, so a tree
  > captured where xattrs exist **cannot be truthfully restored where they don't.**
- **S2:** GC handles ACL'd/`uchg` trees — an ACL of *"everyone deny delete"* made a scratch dir
  **undeletable**; without this C2 strands disk.

**Commit:** `Refuse out-of-substrate entries and emit unrestorable with a named cause`

## Phase 6 — 🔴 The turn gate (M12) — the architectural change

**Goal:** snapshot the **exact** pre-state per turn, without breaking C1's gate.

- **RED:** `tests/test_turn_gate.py` — a `tools/call` is **preceded** by a snapshot; the recorded
  pre-state is the state **before** the call.
- **GREEN:**
  - Add a **`before_frame` hook** to `proxy.run()`. **`proxy.py` must NOT import `json`** — the hook
    is a callback; the parsing lives in `src/belay/sandbox/gate.py`.
  - The gate parses a **COPY** to detect a `tools/call`, **blocks** while snapshotting, then the
    proxy forwards the **ORIGINAL bytes**.
  - **When no sandbox is configured, no gate is installed** and the proxy is exactly C1's byte pump.
- **A10 — NON-NEGOTIABLE:** C1's `test_differential.py`, `test_teeth.py`, `test_import_guard.py`
  **must pass UNTOUCHED**. They assert **bytes**, never timing — which is precisely why C1 wrote the
  claim narrowly.
  > **If the gate breaks byte-identity, the gate is wrong. The gate is not allowed to touch those
  > tests.**
- **Measure and record the added latency.** ~14ms expected. **R-C2-6: publish it; don't hide it.**

**Commit:** `Add the before_frame turn gate: snapshot the exact pre-state`

## Phase 7 — Default scope (M13), threat model (M10), docs

**Goal:** R3-arriving-early, and honesty.

- **M13 zero-config default scope**: the tool's working tree + a sandboxed `$TMPDIR`, so the common
  case needs **no configuration**. **Never silently widen** — a denial is contained and recorded.
- **N1 `belay sandbox check`** — one-shot: does the substrate work here, **and is the scope too tight
  for this server?**
- **M10 `docs/technical/THREAT_MODEL.md`** — Belay runs untrusted actions and **is itself an attack
  surface** (R8). What is enforced · what is not · **what is inexpressible** (per-host network).
- **README coverage update**, stating plainly:
  - **macOS-only.** Linux/Docker is the second slice at Phase-1 packaging. **Linux is entirely
    unverified.**
  - **Seatbelt is deprecated by Apple and functional.** No alternative exists for a spawned CLI.
  - **The gate adds ~14ms per turn** — content-neutral, **not** latency-neutral. *C1 claimed only the
    former, deliberately.*
  - **What we do NOT restore**, each named: birthtime · ctime · atime · ownership-when-not-root ·
    sockets · devices · FIFO contents.
  - **Per-host network allowlists are inexpressible** and are **rejected at the API**.

**Commit:** `Default scope, threat model, and an honest coverage statement`

## Notes for executing agents

- **C2 records. It does not judge.** No PASS/FAIL/UNVERIFIED emitted. C2 *originates* the
  `unrestorable` **fact**; C4 renders the verdict. If you're writing code that decides good/bad —
  stop, wrong capability.
- **Never claim a boundary we don't enforce.** If the substrate can't express it, **reject it at the
  API**.
- **A test that can't fail proves nothing.** Positive controls, ablation, field attribution.
- **Zero runtime deps.** `ctypes` is stdlib. C1's import guard must keep passing.
