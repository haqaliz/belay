## 0. Ground truth (read, not assumed)

- `persist_snapshot(snap, manifest_path)` writes the manifest payload
  (`persist.py:98-116`): `handle, tree_path, backend, capabilities, fidelity_gaps, sidecar`.
  **No source-root field.** Called from `gate._snapshot_turn` at `gate.py:429` with
  `self._scope` (the realpath'd original workspace, `gate.py:298`) in scope but not passed.
- `load_snapshot(manifest_path)` (`persist.py:119-148`) reconstructs `GuardedSnapshot` from
  the payload; builds `Manifest(backend, capabilities, handle)`.
- `Manifest` is a frozen dataclass (`substrate.py:347-357`): `backend, capabilities,
  handle=""`. Adding a defaulted field keeps every positional/keyword caller valid.
- `replay_turn` consumes the loaded `GuardedSnapshot` — so exposing the root on `Manifest`
  makes it reachable at replay with no new plumbing.
- Corpus ingestion copies `manifest.json` verbatim into `<case>/`, so a payload field rides
  along into corpus cases for free (verify in a test).
- **This aspect changes NO verdict.** It records, reads back, and exposes the root — unused.

## 2. Phases (each: RED → GREEN → REFACTOR, suite green, one commit)

### Phase 1 — `Manifest.source_root` + persist/load round-trip

- **Goal:** the manifest carries the realpath'd original root; round-trips; backward + forward
  compatible.
- **Design:**
  - `substrate.py`: add `source_root: Optional[str] = None` to `Manifest` (frozen, defaulted
    last so existing constructions are unaffected). Update its docstring.
  - `persist.py` `persist_snapshot(snap, manifest_path, *, source_root: Optional[StrPath] =
    None)`: when given, write `payload["source_root"] = str(Path(source_root))` **already
    realpath'd by the caller** (the gate passes `self._scope`, which is resolved). Omit the
    key entirely when `None` (keeps old-shaped manifests byte-clean).
  - `persist.py` `load_snapshot`: read `payload.get("source_root")`. **Fail-closed:** if the
    key is present it must be a non-empty **absolute** string, else a named `ValueError`
    (distinguish absent → `None` from malformed → error), matching the loader's existing
    strictness idiom. Populate `Manifest(..., source_root=...)`.
- **Tests (write first):** spec criteria 1–4 —
  `test_manifest_records_the_source_root` (persist with a root → load → `manifest.source_root`
  equals it), `test_manifest_without_source_root_loads_as_none`,
  `test_malformed_source_root_is_a_named_error` (present, blank / relative),
  `test_source_root_realpath_is_the_callers_responsibility` (persist stores the string as
  given; the gate is what resolves — assert persist doesn't silently transform, and that the
  gate path passes a resolved one; keep the realpath contract explicit).
- **Validation:** `uv run pytest tests/test_persist_source_root.py`
- **Commit:** `snapshot: record the source workspace root in the manifest (unused)`

### Phase 2 — gate writes it, and it survives to replay + corpus

- **Goal:** real captures record the root; it reaches `replay_turn` and corpus cases; zero
  verdict change.
- **Design:**
  - `gate.py:429`: `persist_snapshot(snap, self.manifest_root / f"{...}.json",
    source_root=str(self._scope))`. `self._scope` is already `Path(scope).resolve()`
    (`gate.py:298`) — realpath'd, satisfying the prefix-match consistency the relocation
    aspect needs.
  - No change to `replay_turn` logic — it already loads the snapshot; `manifest.source_root`
    is now populated and simply available. Add nothing that consumes it.
- **Tests (write first):**
  - `test_gated_capture_manifest_carries_the_scope` (darwin-gated, mirrors
    `test_phase0_e2e.py` setup): a real gated snapshot's manifest has `source_root` ==
    the resolved scope.
  - `test_replay_turn_sees_source_root_but_verdict_is_unchanged`: replay a fixture capture
    with a manifest that has `source_root` set and one without — **verdicts byte-identical**.
    The no-behavior-change guard (spec criterion 5).
  - `test_corpus_case_manifest_preserves_source_root`: an ingested case's copied
    `manifest.json` still carries the field (rides along free; assert it).
- **Validation:** `uv run pytest`; darwin-gated test runs on this box, skips elsewhere.
- **Commit:** `sandbox: gate records the resolved workspace scope into each turn manifest`

## 4. Testing strategy

- Phase 1 fully offline/deterministic (no sandbox): construct a `GuardedSnapshot`, persist,
  load, assert — mirror `test_persist_relative_tree.py`'s style.
- Phase 2's real-capture test is **darwin-gated** like `test_phase0_e2e.py` (needs Seatbelt);
  the verdict-unchanged and corpus tests are cross-platform (they replay fixtures / inspect
  JSON).
- Commands: `uv run pytest tests/test_persist_source_root.py`, then `uv run pytest`, then
  `uv run ruff check src tests` + `uv run mypy` on touched modules.

## 5. Determinism & verdict impact

- **Verdict impact: none.** The field is recorded and exposed, never consumed. Every existing
  PASS/FAIL/UNVERIFIED is byte-identical — enforced by
  `test_replay_turn_sees_source_root_but_verdict_is_unchanged`. This is the whole safety
  argument for landing the prerequisite independently.
- No clock/network/randomness introduced.

---

## 6. Edge cases

| Case | Behavior |
|---|---|
| Old manifest, no `source_root` | loads `None`, no error (backward compat) |
| `source_root` present but blank / relative | named `ValueError` (malformed ≠ absent) |
| Extra unknown keys in payload | tolerated as today (forward compat) — confirm loader doesn't reject |
| `persist_snapshot` called without `source_root` (non-gate callers) | writes an old-shaped manifest; `None` on load |

**Verdict for a rootless/absent field: still whatever it is today** — this aspect never turns
absent into a verdict. The `UNVERIFIED`-on-rootless honesty rule is the *relocation* aspect's
job, not this one.

---

## 7. Agent execution notes

- **One agent, sequential Phases 1→2** (Phase 2 depends on Phase 1's field). Small, tightly
  scoped; no parallelism needed.
- **Checkpoint after Phase 1:** the manifest schema change is the durable contract — confirm
  the field name/shape before the gate writes it.
- **Ask before proceeding if:** the manifest loader has stricter unknown-key handling than
  assumed (would affect forward-compat), or if any non-gate `persist_snapshot` caller needs
  the root too.
- **Guardrail:** if anything here starts *consuming* the root to change a verdict, STOP —
  that's the `replay-relocation` aspect; this one must stay verdict-neutral.
