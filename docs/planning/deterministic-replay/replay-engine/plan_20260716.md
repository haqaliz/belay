## Phase 1 — 🔴 Snapshot persistence (THE CRITICAL PATH, RED first)

**Goal:** M1 + A6 + A5. Nothing replays until a snapshot survives its process. **Built first for that
reason, and it carries a pre-loaded false PASS.**

- **RED:** `tests/test_snapshot_persist.py`
  - **A6:** persist a `GuardedSnapshot`, then in a **fresh interpreter** (subprocess, or a function
    that takes only the on-disk manifest path) resolve `handle → restorable snapshot` and restore it,
    `BTH-1(restored) == BTH-1(original)`.
  - **🔴 A5 — THE GUARD:** persist, then **synthesize an EMPTY sidecar** and restore from it →
    the BTH-1 comparison **MUST FAIL**, naming `link_group`/`perm`/`mtime_ns`. *This proves the false
    PASS is catchable. Without it, an empty-sidecar C3 ships green.*
- **GREEN:** `src/belay/replay/persist.py`
  - Write a manifest JSON next to the snapshot tree: `{handle, tree_path, backend, capabilities,
    fidelity_gaps}`.
  - **Serialize the sidecar** (`link_groups`, `special_modes`, `dir_times`) as **base64 of the raw
    bytes** (`clone.py:116-130` — *the bytes are the whole point; never decode*).
  - `load_snapshot(manifest_path) -> GuardedSnapshot` reconstructs a **real** sidecar.
  - **Wire it into the gate's snapshot step** so a proxy run leaves resolvable snapshots on disk.
    (Minimal touch to `gate.py`/`substrate.py` — additive; C2 tests stay green.)
- **Validation:** `uv run pytest tests/test_snapshot_persist.py`
- **Commit:** `Persist snapshots so a later process can restore them`

## Phase 3 — The minimal stdio client (RED first)

**Goal:** M3 + M5 + R-C3-3 + R-C3-5.
- **RED:** send a recorded `initialize`+`tools/call` to a fake server via `launch.contained`, read the
  matching reply by id; assert **live state untouched** (replay runs against a scratch restore);
  assert a no-id turn **times out** rather than hanging.
- **GREEN:** `src/belay/replay/client.py`
  - `with launch.contained(server_argv, workspace=scratch, network=deny_all) as spawn:` →
    `Popen(spawn.argv, ...)`; write `frame+b"\n"`; read newline-delimited stdout until the id matches;
    **hard timeout**.
  - Correlate on `(direction, type(id).__name__, id)` — never bare id.
  - **Kill+respawn per replay** (avoids `PROCESS_INTERNAL_STATE`).
- **Commit:** `Add a minimal stdio MCP client for re-invoking a recorded turn`

## Phase 4 — Single-turn replay + the state delta (RED first)

**Goal:** M4 + M6 + M9 + M10 + A1 + A2 + A8.
- **RED:** A1 clean run → result-equivalence; A2 unrestorable pre-state → `unverified`, no result;
  A8 no-handshake trace still replays.
- **GREEN:** `src/belay/replay/engine.py`
  - `replay_turn(records, n)`: read `status` first (M10); `absent`→ can't verify but not failure;
    `unrestorable`→ `unverified` + the recorded cause (**`SNAPSHOT_FAILED` is NOT an
    `UnrestorableCause` — do not coerce it**, `gate.py:143-156`); `present`→ resolve+restore (Phase

    (Phase 3), `scan_tree` before/after → `diff_records`.
  - Key on `tool_calls(index)`, **never `state_handle`** (M9). FIDELITY_GAPS are **not** blocks (M10).
  - Version drift (recorded vs re-negotiated) is a **recorded finding** (M4), not an error.
- **Commit:** `Replay a single recorded turn and capture the state delta`

## Phase 5 — Nondeterminism detection (RED first)

**Goal:** M7 + M8 + A3. **This is where "timestamp diff as FAIL" must be impossible.**
- **RED:** a clock tool and a random tool, replayed **N=3** times, are marked **nondeterministic** —
  NOT reported as a divergence/FAIL. A deterministic tool is **not** so marked (the positive control).
- **GREEN:** `src/belay/replay/determinism.py`
  - Replay N times; divergent results across runs → **nondeterministic**. Append a **`nondeterminism`**
    record (M7 — C1 reserved the kind at `trace.py:52`). **Append to a sibling replay artifact; never
    mutate the source trace** (A4).
  - Raise C3's three inherited causes where they apply (M8); **enum invariant intact** — 18 causes, 0
    unaccounted.
  - **N=3 configurable.** N=2 can't tell "differs" from "differs *consistently*".
- **Commit:** `Detect nondeterminism by repeated replay, appended not rewritten`

## Notes for executing agents
- **C3 observes. It does not judge.** No PASS/FAIL/UNVERIFIED-as-verdict; C3 emits `unverified` as an
  *observation* (can't-be-checked) and the deltas A2 reduces. If you write a verdict — wrong capability.
- **Never let a timestamp diff render as FAIL.** Nondeterminism ≠ infidelity — the distinction is made
  by repeated replay, not by guessing.
- **Never synthesize an empty sidecar** (A5 exists to catch it).
- **`state_handle: present` ≠ replayable turn.** Key on `tools/call`.
- **`UnrestorableCause(cause_string)` throws on `SNAPSHOT_FAILED`.** Don't coerce.
- Zero runtime deps. C1+C2's 250 tests + C1's differential/teeth/import-guard must pass untouched.
